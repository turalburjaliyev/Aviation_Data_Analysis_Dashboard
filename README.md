# Aviation Data Analysis Dashboard ✈️📊

## 📌 Project Overview
This project is an end-to-end data analytics solution focused on the aviation industry. It covers the full data pipeline from web scraping and data processing to database management and interactive dashboard visualization.

The objective is to analyze airline companies, their fleets, operational costs, and aircraft characteristics to generate meaningful business insights.

---

## ⚙️ Project Workflow

### 1. Data Collection (Python)
- Scraped aviation-related data from multiple online sources
- Collected data about:
  - Airlines
  - Fleet size and aircraft types
  - Aircraft specifications (engine power, capacity, fuel consumption)
  - Revenue and operational costs

### 2. Data Processing (Python)
- Cleaned and transformed raw data
- Normalized datasets
- Structured data into relational format
- Generated **16 structured tables**

### 3. Database Integration (SQL)
- Loaded cleaned data into SQL database
- Designed relational schema
- Created **10 analytical queries**
- Generated **26 total tables (raw + query outputs)**

### 4. Data Visualization (Power BI)
- Built an interactive dashboard with **7 pages**, including:
  - Cover page
  - Fleet analysis
  - Financial analysis
  - Aircraft performance
  - Final insights page

---

## 📊 Dashboard Features
- KPI tracking (revenue, cost, fleet size)
- Aircraft comparison analysis
- Airline performance evaluation
- Interactive filtering and navigation
- Final insights and conclusions
